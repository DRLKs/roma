
# Relaciones UML en Rust

- Herencia: 
Inexistente

- Interfaces:
Se pueden desarrollar mediante traits

- Clases abstractas:
Se pueden desarrollar mediante traits y con métodos por defecto.

- Asociación
Son simplemente los campos de una Struct

- Agregación:
Ownership parcial del elemento dentro de la Struct

-Composición:
Ownership completa del elemento dentro de la Struct
